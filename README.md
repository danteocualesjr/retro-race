# Retro Racing Game

A classic 2D top-down retro racing game built with HTML5 Canvas and vanilla JavaScript.

## Features

- **Classic Arcade Racing**: Top-down view racing with retro pixel art aesthetic
- **Multiple AI Opponents**: Race against 3 AI-controlled opponents
- **Power-ups System**: Collect and use power-ups including:
  - Speed Boost: Temporary speed increase
  - Shield: Temporary invincibility
  - Nitro: Instant acceleration boost
  - Slow Motion: Slow down opponents
- **Lap System**: Complete 3 laps to finish the race
- **Scoring System**: Earn points for speed, position, overtaking, and power-ups
- **High Scores**: LocalStorage-based high score tracking
- **Sound Effects**: Retro-style audio feedback
- **Responsive Design**: Adapts to different screen sizes

## Controls

- **Arrow Keys** or **WASD**: Accelerate, brake, turn left/right
- **Space**: Use collected power-up
- **ESC**: Pause/Resume game
- **Enter**: Confirm menu selections

## How to Play

1. Click "START RACE" from the main menu
2. Complete 3 laps around the track
3. Collect power-ups to gain advantages
4. Overtake opponents for bonus points
5. Finish as fast as possible for the highest score!

## Scoring

- **Speed**: Points based on your current speed
- **Position**: Bonus points for higher positions
- **Overtaking**: Points for passing opponents
- **Power-ups**: 100 points per power-up collected
- **Laps**: 500 points per completed lap
- **Time Bonus**: Faster times earn more points
- **Position Bonus**: Higher finishing position earns more points

## Installation

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. No build process or dependencies required!

## Browser Compatibility

Works in all modern browsers that support:
- HTML5 Canvas
- ES6 JavaScript
- Web Audio API
- LocalStorage API

## Technical Details

- Pure vanilla JavaScript (no frameworks)
- Modular code structure
- Object-oriented design
- Efficient collision detection
- Frame-independent physics
